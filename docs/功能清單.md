## 12. 工具函數庫

### DOM 工具
```javascript
DOM.select()      // 選擇元素
DOM.selectAll()   // 選擇多個
DOM.create()      // 建立元素
DOM.addClass()    // 新增類名
DOM.removeClass() // 移除類名
DOM.toggleClass() // 切換類名
```

### 事件系統
```javascript
Events.on()       // 綁定事件
Events.off()      // 解綁事件
Events.trigger()  // 觸發事件
Events.debounce() // 防抖
Events.throttle() // 節流
```

### 儲存系統
```javascript
Storage.get()     // 讀取
Storage.set()     // 寫入
Storage.remove()  // 刪除
Storage.clear()   // 清除
```

### Toast 通知
```javascript
Toast.show()      // 顯示
Toast.success()   // 成功
Toast.error()     // 錯誤
Toast.warning()   // 警告
Toast.info()      // 信息
```

### 模態框
```javascript
Modal.show(id)    // 顯示
Modal.hide(id)    // 隱藏
Modal.init()      // 初始化
```

### 表單工具
```javascript
Form.validateEmail()    // 驗證郵箱
Form.validateRequired() // 驗證必填
Form.getFormData()      // 獲取數據
Form.clearForm()        // 清空表單
```

### 動畫工具
```javascript
Animation.fadeIn()    // 淡入
Animation.fadeOut()   // 淡出
Animation.slideDown() // 向下滑動
Animation.slideUp()   // 向上滑動
```
