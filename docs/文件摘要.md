## 3. 設計指南.md

### 概述
Glassmorphism 2026 設計規範。

### 設計哲學
> "Faster, Smarter, More Inclusive, and More Human"

三大原則:
1. Human Layer - 人性化設計
2. Strategic Minimalism - 戰略性極簡
3. Purposeful Motion - 有目的性動態

### Glassmorphism 核心
```css
background: rgba(255, 255, 255, 0.25);
backdrop-filter: blur(16px);
border: 1px solid rgba(255, 255, 255, 0.4);
border-radius: 16px;
```

最佳實踐:
- 模糊度: 10-30px
- 透明度: 10-30%
- 對比度: ≥ 4.5:1
- Safari: 加 -webkit- 前綴

### 色彩系統

**主色**
| 名稱 | 色碼 |
|------|------|
| Vibrant Purple | #A855F7 |
| Electric Cyan | #06B6D4 |
| Neon Green | #22C55E |

**淺色主題**: #FFFFFF / #F9FAFB / #F3F4F6
**深色主題**: #0F172A / #1E293B / #334155

### 字體系統
- 標題/內文: Inter, Noto Sans TC
- 等寬: JetBrains Mono
- 流體大小: clamp(base, preferred, max)

### 間距系統
0.25rem → 0.5rem → 1rem → 1.5rem → 2rem → 3rem → 4rem

### 圓角系統
8px (sm) → 12px (md) → 16px (lg) → 24px (xl) → 9999px (full)

### 動效系統
- 時長: 150ms (fast) / 300ms (normal) / 500ms (slow)
- 緩動: cubic-bezier(0.4, 0, 0.2, 1)
- 減少動畫: @media (prefers-reduced-motion: reduce)

### 響應式斷點
640px (sm) → 768px (md) → 1024px (lg) → 1280px (xl) → 1536px (2xl)
