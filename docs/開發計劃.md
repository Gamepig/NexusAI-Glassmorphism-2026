## Phase 2: Glassmorphism 核心樣式

### 2.1 玻璃效果組件
```css
.glass-card {
  background: var(--glass-bg);
  backdrop-filter: blur(var(--glass-blur));
  -webkit-backdrop-filter: blur(var(--glass-blur));
  border: 1px solid var(--glass-border);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-md);
}

.glass-nav {
  background: rgba(255, 255, 255, 0.7);
  backdrop-filter: blur(20px);
  border-bottom: 1px solid rgba(255, 255, 255, 0.3);
}

.glass-btn {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(8px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  transition: all var(--duration-normal) var(--ease-smooth);
}
```

### 2.2 佈局系統（Organic Flow，非 Bento）
```css
.container { max-width: 1280px; margin: 0 auto; padding: 0 var(--space-4); }

.sidebar-layout { display: grid; grid-template-columns: 280px 1fr; gap: var(--space-6); }
```

## Glassmorphism 最佳實踐

1. **模糊度**: 10-30px，複雜背景用更高模糊
2. **透明度**: 10-30% 白色/深色疊加
3. **對比度**: 確保文字對比度 ≥ 4.5:1
4. **邊框**: 1px 半透明邊框增加深度
5. **適度使用**: 不要過度使用，保持清晰層級
